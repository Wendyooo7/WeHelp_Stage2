# [台北一日遊](http://54.84.151.83:8000/)
這是一個能讓使用者「探索雙北捷運沿線特色景點」與「預約付費導覽服務」的旅遊電商網站。
<br/>
<br/>
前端 UI 依照 WeHelp Bootcamp 提供的 Figma 文件進行開發，後端依照 WeHelp Bootcamp 提供的景點資料和 Swagger UI API 文件進行開發。　
<br/>
<br/>
